A. JS
------
1. Создайте кнопку с классом red.
   В CSS задайте этому классу property color:red.
   Используйте classList.toggle('red'), чтобы
   при нажатии на кнопку ее буквы то краснели, то 
   возвращались к начальному цвету.

2. Создайте кнопку с классом fantasy.
   В CSS придайте этому классу сказочный шрифт величиной 7vh
   и создайте еще один класс с фонтом другого вида и величины.
   Используйте classList.replace() для того, чтобы
   при нажатии на кнопку у нее менялся шрифт.

3. Вам задан текст:
   var alphabeth = 'abcdefghijklmnopqrstuvwxyz'.
   Превратите его в массив.
   Превратите его обратно в string так, чтобы все буквы 
   шли через запятую.
